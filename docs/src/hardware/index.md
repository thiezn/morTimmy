# Hardware

This section covers the current robot hardware from three angles:

- [Design](design.md): system architecture, power distribution, and design rules.
- [Components](components.md): concrete parts in use, with one page per component.
- [Wiring](schematics/wiring.md): the generated wiring diagram and related schematic outputs.

The goal is to keep high-level rationale, part-level documentation, and concrete wiring artifacts separate but cross-linked.
