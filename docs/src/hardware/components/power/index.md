# Power System Design

We will eventually navigate towards a separate PCB for a power board but for the initial prototype we use:


- battery holder for two 18650 3000mAh rechargable li-ion 3.7V
- 5V 2A buck converter (for the Raspberry Pi)

- DIY 2-Slot 18650 Battery Holder with Pins – Black
- AC Charger + 2xUltraFire 18650 3.7V 3000mAh Rechargeable Battery
- 3A-6S UBEC 'hobbywing' for powering the Raspberry Pi and stabilizing the voltage
